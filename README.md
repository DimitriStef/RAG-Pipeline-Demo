# RAG-Pipeline-Demo

A lightweight agentic Retrieval-Augmented Generation (RAG) project that summarises topics extracted from web sources.

Current features:
- Basic web crawling and text cleaning for corpus creation
- Embedding generation, and storage in ChromaDB
- Extractive QA and NLI for autonomous web crawling
- TinyLlama SLM for answer generation
- LangChain-based retrieval and orchestration
- Streamlit interface for interactive querying
